# laravel-vue-composition-vuetify-skeleton
Laravel Vue Skeleton es una aplicación base con Laravel 12, Vue 3.5 (Composition API), Inertia.js y Vuetify 3. Incluye autenticación (login, registro, recuperación de contraseña, verificación de email) y gestión de perfil de usuario. Utiliza Vite 6 como herramienta de build y está configurado para desarrollo con soporte en español.
